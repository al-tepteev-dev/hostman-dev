# Hostman Code Showcase

Этот репозиторий содержит фрагменты кода, написанные мной во время работы в компании [Hostman](https://hostman.com/)(Дочки https://timeweb.com). Здесь представлены компоненты и страницы, разработанные для основного веб-сайта.

## О проекте

Код в этом репозитории является частью большого проекта и демонстрирует мои навыки в разработке на Nuxt/Vue 3 + TypeScript ,SCSS-modules

## Стек технологий

- **Framework**: Vue.js (Nuxt.js)
- **Язык**: TypeScript
- **Стили**: SCSS
